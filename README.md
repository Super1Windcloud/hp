# hyperscoop    (WIP 正在开发中)

## 🐼 一个更快,更强, 更精美的  windows 包管理器基于 rust ( 继承自 scoop )

## 快速开始

```bash
 scoop bucket add hp https://gitee.com/SuperWindcloud/scoop-hyperscoop.git
scoop  install  hp/hyperscoop 
 
 ```
